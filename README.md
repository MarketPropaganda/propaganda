# propaganda
propaganda 优逸客宣传项目，是一个面向学员宣传推广优逸客教学优势的离线项目，主要模块包含：公司介绍、web全栈开发实训、ui/ux设计实训、web优秀作品展示、ui优秀作品展示、学员就业明星、教学保障。
该项目主要采用了相册类型的展示方式，框架基础主要来源于Bret Victor的作品，首先对原作者表示敬意！！
项目内文件目录结构简介，此处忽略常规目录主要说明一下资源分配目录：
    ThumbnailImages   缩略图存放目录(格式要求.jpg)
    PageImages           展示图存放目录(格式要求.jpg)
    Page                      展示页存放目录
    Media                    展示视频存放目录(格式要求.webm请不要放过大视频)
以上目录下又加入对应模块目录：
    CompanyIntroduction    公司介绍目录
    TrainForWEB                  web全栈开发实训目录
    TrainForUI                      ui/ux设计实训目录
    ProjectForWEB               web优秀作品展示目录
    ProjectForUI                   ui优秀作品展示目录
    StarStudents                  学员就业明星目录
    EmploymentSecurity       教学保障目录
